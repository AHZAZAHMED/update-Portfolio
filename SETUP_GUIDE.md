# Portfolio Project Setup Guide

## Prerequisites
- Node.js 18+ (✓ Already installed)
- Bun (recommended) OR npm/pnpm

## Quick Start

### Option 1: Using Bun (Recommended - Faster)

1. **Install Bun** (if not installed):
   ```powershell
   powershell -c "irm bun.sh/install.ps1|iex"
   ```

2. **Install dependencies**:
   ```bash
   bun install
   ```

3. **Run development server**:
   ```bash
   bun run dev
   ```

4. **Open browser**: http://localhost:3000

---

### Option 2: Using npm

1. **Remove Bun lock file**:
   ```bash
   rm bun.lock
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Open browser**: http://localhost:3000

---

## Customization

### Update Personal Information

Edit these files to customize the portfolio:

1. **src/routes/index.tsx** - Page metadata and structure
2. **src/components/Hero.tsx** - Name, roles, tech stack
3. **src/components/Sections.tsx** - Projects, skills, education, contact info
4. **src/components/AIHub.tsx** - AI assistant knowledge base
5. **src/assets/profile.jpg** - Replace with your photo

### Change Colors/Theme

Edit **src/styles.css** - CSS variables for colors:
- `--background` - Main background color
- `--foreground` - Text color
- `--neon` - Accent color (green)
- `--cyan-accent` - Secondary accent

## Deployment

### Deploy to Cloudflare Pages (Recommended)

1. **Build the project**:
   ```bash
   bun run build
   ```

2. **Deploy with Wrangler**:
   ```bash
   npx wrangler pages deploy .output/public
   ```

### Deploy to Vercel/Netlify

The project is SSR-ready and can be deployed to any platform supporting Node.js.

## Troubleshooting

### Port already in use
If port 3000 is busy, Vite will automatically try the next available port.

### Dependencies installation fails
- Try deleting `node_modules` and lock files, then reinstall
- Make sure you're using Node.js 18 or higher

### Build errors
- Run `bun run lint` to check for code issues
- Check that all imports are correct
